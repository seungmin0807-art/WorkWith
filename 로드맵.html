Phase 2 · 핵심 기능 구현
6
라이브 카메라 실시간 분석 무료
MediaPipe Pose + 브라우저 카메라 API로 아이폰 카메라에서 실시간 포즈 감지. mp4 파일 방식 → 라이브 스트림 방식 전환.
7
세션 결과 저장 & 히스토리 화면 무료
운동 세션 결과를 IndexedDB에 저장. 날짜별 히스토리 리스트와 점수 추이 그래프 화면 구현.
8
음성 코칭 고도화 무료
Web Speech API 자연스러운 음성 개선. 실시간 카운트 + 교정 큐 타이밍 최적화. 한국어/영어 전환.
9
설정 화면 무료
경고 점수 임계값 커스텀, 언어 설정, 운동 알림 (Web Notification API). LocalStorage 기반 설정 저장.
Phase 3 · 앱 품질 향상
10
모바일 60fps 성능 최적화 무료

11
앱 아이콘 & iOS 스플래시 풀셋 무료
